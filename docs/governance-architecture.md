# Governance architecture

Governance controls are linked directly to evaluation signals.

| Evaluation signal | Governance control |
| --- | --- |
| Missed retrieval | Evidence map review and retrieval improvement backlog |
| Wrong citation | Citation support gate and citation incident handling |
| Unsupported answer | Refusal quality checks and human review |
| Multi-hop failure | Baseline limitation tracking and Stage 4 release notes |
| Safety flag | Human review queue and incident response playbook |
| Human review mismatch | Human review protocol and reviewer audit trail |
| Rubric score weakness | Dimension-specific improvement plan |

The aim is not to hide weak scores. Weaknesses are kept as evidence for prioritising future improvements.
