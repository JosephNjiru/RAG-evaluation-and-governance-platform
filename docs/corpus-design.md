# Corpus design

The Stage 1 corpus is synthetic and public-safe. It contains ten Markdown documents across organisational AI policy, data governance, security, monitoring, incident response, analytics, RAG evaluation, privacy, human review and knowledge base maintenance.

Each source document includes:

- document ID
- title
- version
- date
- owner
- risk domain
- section headings
- source text

The corpus is intentionally small so evidence can be inspected manually. It is large enough to support factual, multi-hop, ambiguous, refusal and adversarial evaluation cases.
