# Responsible AI checklist

- Evaluation dataset exists before generation.
- Answerable questions have evidence map rows.
- Unsupported questions do not have fabricated support.
- Adversarial questions are labelled.
- Retrieval and citation metrics compare against expected evidence.
- Refusal behaviour is tested.
- High-risk and failed records enter human review.
- Limitations are documented.
- No paid API is required for default checks.
- Release scan checks private planning language, secret-like strings and inflated claims.
