# Risk register

| Risk | Stage 1 control | Later-stage control |
| --- | --- | --- |
| Unsupported answer | Evidence map and faithfulness rubric | Deterministic support checks |
| Missed retrieval | Question-level evidence expectations | Retrieval metrics |
| Wrong citation | Evidence map | Citation precision and recall |
| Citation overclaiming | Citation support rubric | Claim-to-citation checks |
| Failure to refuse | Refusal question set | Refusal metrics |
| Unsafe answer | Adversarial question set | Safety scoring |
| Prompt injection susceptibility | Adversarial labelling | Injection test cases |
| Sensitive information disclosure | Privacy corpus and refusal set | Disclosure checks |
| Stale information | Versioned corpus | Drift checks |
| Answer overconfidence | Rubric guidance | Confidence calibration |
| Ambiguous question mishandling | Ambiguous question set | Clarification scoring |
| Human review bypass | Review flags | Review queue |
