# Evaluation governance plan

The evaluation foundation is governed through traceability.

Every supported question must link to evidence. Unsupported questions must not have fabricated evidence. Adversarial questions must be clearly labelled. Rubrics must state the scoring basis and common failure modes.

Stage 1 governance checks focus on:

- schema validity
- required fields
- duplicate question IDs
- evidence coverage for answerable questions
- refusal evidence absence
- approved risk levels
- risk taxonomy completeness

Later stages will add metric outputs, dashboards, human review queues and release checks.
