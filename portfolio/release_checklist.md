# Release checklist

- Git status checked.
- Ignored file check completed.
- No `.venv` committed.
- No `.git` included in clean archive.
- Tests passed.
- Ruff passed.
- Docker config passed.
- API health check passed.
- Dashboard generated.
- README render checked.
- Mermaid checked.
- No company wording.
- No secret-like strings.
- No `production-ready`.
- No false cloud deployment claim.
- GitHub Actions confirmation after push.
